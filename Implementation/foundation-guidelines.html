Foundation Guidelines

Our color palette includes a core grayscale range along with a set of bright accent colors. While the grayscale and primary accent colors are used most frequently, each color family also includes approved tints and shades. Beneath each block, the corresponding token name and hex value are provided for reference.

Refer to the accessibility guidelines at the bottom of this page for detailed contrast requirements.

Use --color-gray-900 text on the first two rows of the palette. For the remaining rows, use --color-gray-300 text. Only backgrounds using --color-gray-900 may pair with any of the -700 text‑color tokens (excluding gray). See the example image below for approved combinations. This ensures that we stay above the minimum contrast ratio requirements.
